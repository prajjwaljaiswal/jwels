A subdomain request, end to end

  1. Browser opens jhumkaya.store.vrindaonline.com.
  2. Caddy :443 terminates TLS with the wildcard cert, proxies to 127.0.0.1:3003.
  3. middleware reads the Host header → classifies it as subdomain jhumkaya.
  4. middleware → GET /resolve?by=subdomain&key=jhumkaya → { vendorId, slug }.
  5. middleware rewrites / → /jhumkaya into the existing route tree.
  6. The RSC layout's getBrand() calls GET /jhumkaya/brand (cached, tagged).
  7. Theme is merged on the server → themed HTML ships in the first byte. No flash.

PM2-hosted · self-signed nothing — Caddy issues every cert · the same route tree serves path, subdomain, and custom-domain access.
